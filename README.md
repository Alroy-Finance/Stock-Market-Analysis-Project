**Overview**  
This project analyzes historical stock data to evaluate performance, risk, and trends across major technology companies. Using Python, the analysis focuses on returns, volatility, and technical indicators such as moving averages to derive investment insights.

**Objectives**    
Analyze historical stock price data for multiple equities.  
Calculate daily returns and volatility.  
Compare risk-return profiles across stocks.  
Identify trends using moving averages.  
Visualize stock performance over time.

**Stocks Analyzed**  
Apple (AAPL), Tesla (TSLA), Microsoft (MSFT), Google (GOOGL)

**Tools & Technologies**  
Python, Pandas, NumPy, Matplotlib, SciPy, Data Source: Yahoo Finance

**Methodology**  
Data Collection: Retrieved historical adjusted closing prices using Python API.  
Data Processing: Calculated daily percentage returns. Computed average returns and volatility (standard deviation).  
Visualization: Plotted stock price trends for comparison and generated moving averages (50-day and 200-day).  
Technical Analysis: Used moving averages to identify bullish trends and momentum

**Results**  
Average Daily Returns:  
TSLA: 0.00197 (highest return)  
GOOGL: 0.00183  
AAPL: 0.00093  
MSFT: 0.00066  

Risk (Volatility):  
TSLA: 0.0400 (highest risk)  
GOOGL: 0.0191  
AAPL: 0.0176  
MSFT: 0.0140 (lowest risk)
